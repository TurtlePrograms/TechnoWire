History of programming languages





1957

Fortran (IBM)

Designed for scientific and engineering calculations, Fortran was the first high-level programming language.

1958

LISP (John McCarthy)

Known for its symbolic processing capabilities, LISP is often used in artificial intelligence and symbolic mathematics.

1959

COBOL (CODASYL)

Developed for business data processing, COBOL is still used in legacy systems today.

1972

C (Dennis Ritchie)

C is a versatile, low-level language used for system programming, embedded systems, and more.

1995

JavaScript (Brendan Eich)

JavaScript is the primary language for front-end web development, providing interactivity and dynamic content.

1995

Java (James Gosling and Sun Microsystems)

Java is platform-independent and used for building web applications, Android apps, and more.

1995

Ruby (Yukihiro Matsumoto)

Ruby emphasizes simplicity and productivity, known for its elegant syntax.

1990

Haskell (Haskell community)

Haskell is a purely functional programming language with strong type inference.

1991

Python (Guido van Rossum)

Python is known for its readability and ease of use, making it popular for web development, data science, and automation.

2003

Scala (Martin Odersky)

Scala combines object-oriented and functional programming features and runs on the JVM.

2010

Rust (Mozilla Research)

Rust is designed for systems programming with a focus on safety, preventing memory-related bugs.

2009

Go (Golang) (Google)

Go is known for its simplicity and efficiency, making it suitable for concurrent and networked systems.

2012

TypeScript (Microsoft)

TypeScript is a superset of JavaScript that adds static typing, improving code quality and maintainability.

2014

Swift (Apple Inc.)

Swift is designed for iOS, macOS, watchOS, and tvOS app development, with a focus on safety and performance.

2011

Kotlin (JetBrains)

Kotlin is a statically-typed language that runs on the Java Virtual Machine (JVM) and is known for its conciseness and safety features.
