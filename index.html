about us

TechnoWire is a website where you can learn about different programming languages. We are a group of 4 students from the Netherlands. We are currently studying at the ROC van Amsterdam. We are currently in our first year of the study Software Development. We are working on this project for our school. We are doing this project for our school because we need to show that we can work in a group and that we can make a website. We are currently working on this project for 10 weeks. We have learned a lot about programming languages and how to make a website. We hope that you have learned something about programming languages and that you have enjoyed our website.
